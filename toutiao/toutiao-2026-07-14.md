# 中国 AI 大模型 11 周霸榜全球！腾讯 Hy3 (free) 空降登顶 6.13 万亿 Token，DeepSeek 七连冠终结，英伟达 Nemotron 首次登榜

> AI News Daily / 2026-07-14 头条

## 一份看似枯燥的调用榜，藏着中国 AI 产业最关键的信号

7 月 13 日凌晨，OpenRouter 官方悄悄更新了新一周的大模型调用榜。

榜单上没有炫目发布会。没有几十亿融资。只有一堆数字。

但这些数字**是过去 6 个月，全球 AI 产业最重要的实时体检**。

一周之内，**全球所有大模型的调用 Token 量突破了 54.6 万亿**。这个数字环比上一周**增长 12.6%**——继 6 月末破 50 万亿之后，7 月第 2 周继续拉高上限。

更关键的是分区数据：

- **中国 AI 大模型：27.58 万亿 Token**（环比 **＋17.61%**，连续 7 周增长）
- **美国 AI 大模型：6.33 万亿 Token**（环比 ＋47.9%，反弹但基数低）
- **中国是美国的 4.36 倍**

而中国大模型的周 Token 调用量，**已经连续 11 周超过美国**。

## 榜单前六，全是中国队

这一次的洗牌力度前所未见。

**第一名：腾讯 Hy3 (free)，6.13 万亿 Token**——空降登顶，终结了 DeepSeek-V4-Flash **连续七周**的冠军纪录。

**第二名：小米 MiMo-V2.5，5.95 万亿 Token**（**＋36%**）——距榜首只差 0.18 万亿，随时可能反超。

**第三名：DeepSeek-V4-Flash，5.22 万亿 Token**（**-2%**）——七连冠告终，但依然是全球最稳定的中国开源模型基本盘。

**第四名：MiniMax M3，4.26 万亿 Token**（＋4%）。

**第五名：智谱 GLM 5.2，3.19 万亿 Token**（**＋24%**）——发布不满 3 周，已经进入全球前五。

**第六名：阿里 Qwen 4.1 235B，2.87 万亿 Token**（＋19%）。

**前六名，全部为中国 AI 模型。**

这份榜单不是「中国模型排行榜」。它是**面向全球 200 多万开发者的 OpenRouter API 网关**——用户遍布 130＋ 国家。你用哪个模型，用了多少 Token，一切都写在数据里。

## 最亮的黑马：英伟达自研模型首次登榜

如果你以为这是「中国模型的一面倒」，还有一个惊喜。

**英伟达 Nemotron 3 Ultra 首次登榜第八，2.04 万亿 Token，环比 ＋109%**。

- 6 月 4 日刚上线，**不满 6 周即杀入全球前十**。
- 推理成本**仅头部闭源模型的 1/10**。
- 定价挑战 DeepSeek V4 Flash 的 $0.28/M output（对比 GPT-5.5 的 $30/M output）。

**英伟达终于不满足只卖芯片，开始亲自下场做模型。**

这是 Jensen Huang 的一贯风格——**当他决定进入一个市场，通常意味着这个市场的游戏规则要变了**。

阶跃星辰 Step 3.7 Flash 这一周跌出前十榜单——**Step Pro（参数超 1.5 万亿）即将发布**，阶跃需要一场翻身仗。

## 冠军背后的技术账：腾讯 Hy3 到底强在哪

**腾讯 Hy3 混元模型的核心参数**：

- **2950 亿参数 MoE 架构**（激活 210 亿）
- **256K context**——比市面主流大模型多 4-8 倍上下文
- **SWE-bench Pro 得分 57.9%**——反超 DeepSeek V4 Pro
- **腾讯股价发布日单日＋4.82%**

腾讯这次做对了几件事：

1. **命名策略**——「Hy3 (free)」意味着「混元第三代 + 免费」，通过 OpenRouter 免费投放，快速抢占 API 流量。
2. **上下文优势**——256K 上下文让长文档、代码库、多轮 Agent 场景全线覆盖。
3. **Agentic 能力**——SWE-bench Pro 是最难的智能体编码基准，超过 DeepSeek V4 Pro 就是拿到「Agent 时代的入场券」。

腾讯把过去 3 年一直做研发但发布相对低调的混元系列，用一场「免费空投」的方式打到了全球第一。

## 更值得警惕的信号：美国企业已经用中国模型 3 成以上

如果说 11 周连续第一还只是「量」，那真正的「质变」在这里——

**美国企业用中国模型 Token 占比，自 2026 年 2 月起稳定 >30%**，**最高触及 46%**（2025 年上半年仅 4.5%）。

这意味着：

- 中国模型不只是「在开发者社区里免费领先」；
- 它已经**穿透美国企业的付费栈**；
- 硅谷创业公司在用；华尔街量化在用；甚至部分华盛顿智库都在用。

背后的逻辑很朴素——**性能追平，价格差 20-100 倍**。当 DeepSeek V4 Flash 输出定价 $0.28/M 对上 GPT-5.5 的 $30/M，即使性能有 10% 差距，市场也会用脚投票。

## 榜单读法：不是排名，是产业信号

**信号一**：中国大模型的**开发者渗透率**已经领先全球。「谁的 API 被 API 网关调用最多」是最诚实的市场指标——比融资估值真实 100 倍。

**信号二**：**MoE 架构 + 免费投放**成为新一代冠军战术。腾讯 Hy3、小米 MiMo、DeepSeek 全部是 MoE 结构；免费投放（或极低价格）扫射 OpenRouter 已经成为标配打法。

**信号三**：**闭源模型的价格护城河正在崩塌**。英伟达自研 Nemotron 3 Ultra 用「1/10 成本 + 开源可微调」正面挑战闭源阵营——**连英伟达都开始做开源**，是闭源阵营 2026 年下半年最大的警报。

**信号四**：**Agent 时代的算力消费面**远比训练算力面大。54.6 万亿 Token / 周 = 每天调用 7.8 万亿 Token = 全球每人每天贡献 977 个 Token 调用——AI 推理已经进入「大众消费级流量」阶段。
