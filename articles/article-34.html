Anthropic 的临界点：Claude Fable 5 剥离订阅，AI 旗舰模型进入按量计费时代

7 月 1 日 Claude Fable 5 限时回归、订阅用户获 7 天 50% 周额度、7 月 7 日之后彻底剥离订阅转按 usage credits 计费——这是继 6 月 15 日 Agent SDK 剥离方案在最后一刻叫停之后，Anthropic 第一次把真正落地的「剥离」动作放到旗舰模型身上；也是继 7 月 2 日 OpenAI 推理成本腰斩之后，大模型商业化又一次范式撕裂。本文从事实核对、剥离机制、成本经济学、开发者账单、三家横评、监管与地缘 6 个维度，拆解订阅制走到临界点之后的产业选择。

2026 年 7 月的第一周，Anthropic 用一份看起来只有 7 天的「限时优惠」，把一件酝酿了三个月的产业转折做实了：Claude Fable 5 从 7 月 1 日开始向订阅用户开放，但每周只有 50% 额度、7 月 7 日太平洋时间 23:59:59 之后彻底退出所有订阅套餐，转为按 usage credits 计费。API 挂牌价为输入 $10 / 输出 $50 每百万 Token——大约是 Opus 4.8 的两倍。这一步撕开的不是一款模型的定价，而是过去 18 个月「订阅制补贴推理算力」这套商业模式的临界点。

把这一步放到时间轴上更清晰。6 月 15 日，Anthropic 曾计划把 Claude Agent SDK 从 Pro/Max/Team 订阅池整体剥离、迁入独立 Agent SDK Credit 池，方案已经敲定金额（Pro $20、Max 5x $100、Max 20x $200）、日期、实施细则，但在生效当天的邮件里被自己叫停，官方措辞是「重新打磨方案」；仅仅 15 天后，Anthropic 用旗舰模型 Fable 5 完成了同一件事——不再走「所有 SDK 一起剥离」，而是走「按模型级别剥离，从最贵的开始」。这份路径修改的信息量，远比一个促销周期要重。

与此同时，同一周 OpenAI 的推理成本被 The Information 报道下调 50%（本站 7 月 2 日《OpenAI 推理成本腰斩》已做深度分析），DeepSeek 官宣 V4 正式版 7 月中旬上线并首推峰谷定价、白天 7 小时高峰段 API 价格翻倍——三家 AI 头部厂商在同一个窗口做出三种截然不同的选择：OpenAI 用成本下降去支撑订阅制、Anthropic 用剥离旗舰去承认订阅制承载不了尖端模型、DeepSeek 用峰谷把边际算力成本传导给用户。这不是三份独立的新闻，是同一场商业模式重排的三面切片。

一、事实核对：Fable 5 剥离的六项关键事实

产业分析要先把事实钉死。综合 Anthropic 官方博客《Claude Fable 5 promotional access》、Anthropic Help Center 相关条目、36 氪 7 月 2 日《「白月光」回归秒变「吞金兽」》、智东西 7 月 2 日报道、新智元微博 7 月 2 日推送、CSDN 博客 7 月 2 日《Anthropic Agent SDK 计费风波背后的 Agent 经济学焦虑》、Digital Applied 6 月 16 日《Claude Credit Overhaul》以及 Usagebox 6 月 9 日路径分析，本次 Fable 5 剥离的关键事实可以整理如下。

  • 关键时间轴：Fable 5 于 2026-06-09 首发，是 Anthropic 定位 Mythos 级的旗舰模型；6-12 因美国出口管制指令临时下架、6-30 管制解除；7-01 限时回归并同步公告 promotional access 规则；7-07 PT 23:59:59 促销窗口关闭、Fable 5 彻底退出订阅套餐。
  • 限时窗口：Pro / Max 5x / Max 20x / Team 订阅用户在 7-01 至 7-07 期间，可将每周使用额度中 最多 50% 用于调用 Fable 5，其他模型的用量与 Fable 5 共享同一个池；额度未用完部分不结转到下一周。
  • API 定价：输入 $10 / 百万 Token、输出 $50 / 百万 Token，约为 Opus 4.8（$5/$25）的两倍；prompt caching 90% 折扣仍然生效；美国境内运行的工作负载额外加价 1.1 倍，仅在美国基础设施推理。
  • 安全风控升级：Fable 5 引入独立于主模型之外的 safety classifier「多层防护机制」（defense in depth），针对网络安全、生物化学、模型蒸馏三类高风险请求做实时判定；触发后请求会被拦截或自动回退到 Opus 4.8。开发者社区反馈误伤率显著高于 Opus 系列。
  • 消耗速度：Anthropic 官方文档明确 Fable 5「相比其他 Claude 模型消耗额度更快」，实测下单个复杂任务 Token 消耗约为 Opus 4.8 的两倍；Max 20x 用户里已有报告显示 3 个大 prompt 消耗 $90 API 费用、30 分钟耗掉 60% 5 小时段额度。
  • 7-07 之后的路径：Fable 5 从所有订阅套餐里彻底剥离，用户要么开启 usage credits 按 API 挂牌价继续调用，要么切换到 Opus 4.8 / Sonnet 5 继续消耗订阅额度。Claude.ai 网页端、桌面端、移动端交互式使用不受影响。

需要说明的是，本文所有价格与规则以 Anthropic 官网 2026-07-05 展示为准，Enterprise 长约、教育/政府折扣通常有 20-40% 让利，本文不展开；Token 消耗数字与开发者账单来自 36 氪、智东西、Digital Applied、Usagebox 与开发者公开博客的交叉引用，与真实负载相关、有波动性。所有引用均标注公开信源，不做超出信源的推断。

二、剥离机制：从「Agent SDK 剥离」到「按模型剥离」

要理解 7 月 1 日这份 promotional access 的份量，必须先把 6 月 15 日那次被叫停的方案回放一遍。

Anthropic 在 2026-05-14 首次公告：从 6-15 起，Claude Agent SDK、claude -p（非交互式模式）、Claude Code GitHub Actions、以及基于 Agent SDK 的第三方应用（OpenClaw、Conductor、Zed 等）将被整体从 Pro/Max/Team/Enterprise 订阅池剥离，转入独立的 Agent SDK Credit 池：Pro $20/月、Max 5x $100/月、Max 20x $200/月、Team Standard $20/席、Team Premium $100/席。Credit 不跨席位共享、不滚存、月末失效；用完之后要么开启 usage credits 溢出、要么直接拒绝调用。

这套方案的商业逻辑其实一目了然：把「程序化」和「交互式」两种使用彻底分账，去掉订阅池对 Agent SDK 的隐性补贴。据 Digital Applied 6 月 16 日追踪报告，Anthropic 内部估算，重度 Agent SDK 用户的实际算力消耗大约相当于 API 零售价的 15-30 倍订阅费——一份 $20 的 Pro 订阅能撬动 $300-$600 的 API 等值算力。这就是 Sonnet 4.6 + Opus 4.7 发布之后 Anthropic 推理算力持续紧张的根本原因，也是 T3 Code 创始人 Theo Browne 5 月 30 日公开批评「Claude Code 体验大幅降级」的技术底色。

但方案在 6 月 15 日生效当天被自己叫停。Anthropic 给用户的邮件措辞很温和：「Anthropic 正在重新打磨方案。眼下什么都不变——你的订阅额度照旧，没有 Credit 需要领取。」关键在于「重新打磨」四个字——这个方向不会改，只是这次落地方式不合适。

然后是 7 月 1 日的 Fable 5 promotional access。这一次的剥离路径变了三件事：

维度6-15 Agent SDK 剥离（暂停版）7-01 Fable 5 剥离（落地版）
剥离对象所有 Agent SDK / -p / GitHub Actions单一模型 Claude Fable 5
受影响范围程序化调用（Claude Code 交互不变）所有订阅通道（含 Claude.ai）
剥离窗口一步到位7 天促销期 + 50% 额度过渡
价格锚点API 挂牌价（$3/$15 Sonnet、$5/$25 Opus）API 挂牌价 $10/$50（Opus 2 倍）
社区反应Reddit、HN、X 集体抗议「先用为敬」+ 对高价的负面吐槽
商业意图去掉隐性补贴、堵住 SDK 套利为最贵的模型建立独立计费池

从这张对比表里可以看出，7-01 的落地版做了三个关键调整：第一，从「按接口类型剥离」改成「按模型级别剥离」——这样最贵的旗舰模型先走，中低端 Opus 4.8 / Sonnet 5 / Haiku 4.5 依然由订阅池承载；第二，给了 7 天 50% 额度的缓冲——让用户不至于一夜之间账单翻倍，也顺势筛掉了不愿付按量的低意愿用户；第三，把风险最集中的用法从「SDK」重定义到「模型」——这一步比 SDK 剥离更精准，因为 SDK 剥离误伤了太多轻度自动化用户，而按模型剥离只影响到 Fable 5 这一款尖端需求。

「Anthropic 没有放弃为程序化用量重新定价的方向，只是把这次落地按下了暂停键，并承诺下次会给足提前量……方向不会变，只是路径会变。」

—— CSDN 博客《Anthropic Agent SDK 计费风波背后的 Agent 经济学焦虑》，2026-07-02

三、成本经济学：为什么旗舰模型必须先剥离

为什么是 Fable 5 先走、而不是 Opus 4.8 先走？答案在推理成本的边际曲线上。

本站 7 月 2 日《OpenAI 推理成本腰斩》已经拆过 2026 上半年推理成本的普遍性下降——纯软件层优化让 OpenAI 的推理毛利率提升到 60%+、Anthropic 的毛利率在 18 个月里从 -94% 拉回到 60%+。这份普遍性下降有一个前提：模型架构与训练数据分布相对稳定。一旦上到 Mythos 级模型（Fable 5 的定位），架构和推理路径都会重构，Token 单价与激活参数量重新爆炸。

Anthropic 官方文档承认 Fable 5「消耗额度更快」，实测下单个复杂任务 Token 消耗约为 Opus 4.8 的 两倍。这意味着，即使 Fable 5 的 API 挂牌价是 Opus 4.8 的 2 倍，用户单位任务的总账单其实是 Opus 4.8 的 4 倍（价格 ×2 × Token ×2）。这个 4 倍数字，就是订阅池承载不了 Fable 5 的核心原因——把 4 倍成本的模型硬塞进原本按 Opus 4.8 补贴的订阅池里，唯一结果是订阅池亏光，Anthropic 推理毛利率会被 Fable 5 单一模型拖回负值。

做一份反向估算：以 Max 20x（$200/月）为例，原本一个月 200 万 Token（假设按 Opus 4.8 输入 $5/输出 $25、3:1 混合）可以吃掉约 $30 API 等值；剥离前用户实际能跑的量是这个数字的 15-30 倍——大约 $450-$900 的 API 等值算力。如果 Fable 5 走同样的补贴，因为 Token 消耗和挂牌价都是 2 倍，用户能撬动的 API 等值算力会飙到 $1800-$3600，也就是 Anthropic 每挣 $200 订阅费就要亏 $1600-$3400。这份账，怎么算都是不可持续的。

剥离的经济学结论其实很朴素：推理算力软件层优化的成果，只能承担「主流模型」的普及，不能承担「尖端模型」的普及。这也是为什么 7 月 1 日之后，Anthropic 会明确告诉用户「大多数编码任务不受影响」——大多数编码任务其实用 Opus 4.8 / Sonnet 5 就能跑，只有 Fable 5 才是真正的尖端需求。

四、开发者账单：谁被撬动、谁被留下

剥离一定会撬动一批用户。根据开发者社区反馈与公开博客数据，Fable 5 剥离最直接影响到三类用户：

  • 高频 CI/CD 用户：每天跑数十次自动化任务，一个月轻松消耗 5000 万-1 亿 Token。按新定价，$200 Max 20x 只够覆盖约 1300 万 Token 的 Fable 5 用量——超额部分要么按 API 挂牌价买单、要么切回 Opus 4.8 妥协。这类用户面临「重度 agent 使用」+460% 涨幅（据开发者社区对比表）。
  • 复杂多步 Agent 用户：单次会话消耗 50 万-100 万 Token。这类用户如果坚持用 Fable 5，一个月账单会飙到 $580-$1120（对比订阅费涨幅 190%-460%）。多数用户会被迫回到 Opus 4.8 —— 但 Opus 4.8 的复杂多步能力显著弱于 Fable 5。
  • 科研与专业分析用户：如癌症疗法、生物化学分析这类需求，Fable 5 触发 safety classifier 的概率显著高于 Opus，请求会被拒绝并自动回退到 Opus 4.8。教授 Derya Unutmaz 在 X 上公开表示「Fable 5 回来了，还是跟以前一样没用」。BridgeBench 上 Fable 5 因拒绝回答的比例上升，排名从第 9 名跌到第 41 名。

而被留下来的，是三类价值最集中的付费用户：

  • 企业级用户：接受 usage credits + Enterprise 长约，把 Fable 5 用在真正需要旗舰模型的场景（法务分析、复杂财务建模、高端咨询）。这类用户不在乎单次账单，只在乎结果质量。
  • Project Glasswing 大客户：Anthropic 面向 Apple、Microsoft、Google、Nvidia、AWS、JPMorgan 提供的 $100M 使用额度包，Mythos Preview 定价为 Opus 4.6 的 5 倍——这些客户的 Fable 5 消耗被单独计费，与订阅池完全隔离。
  • API 直连开发者：跳过订阅制、直接接 Anthropic API，把 Fable 5 用在 SaaS 产品的定价里加价传导给下游客户。

剥离的净效果是——Fable 5 从「订阅制补贴 + 全民可用」转成「按量付费 + 企业专属」。这份分层，短期让 Anthropic 承担了「傲慢」的社区评价（Reddit 上有开发者写道：「他们先说自己做出了史上最好的模型，然后又说『哈哈开个玩笑，这模型太危险了，所以你不能问这些问题』」），但换来了旗舰模型的商业可持续性。

五、三家横评：Anthropic / OpenAI / DeepSeek 的三种答卷

把镜头拉到整个行业，2026 年 6 月末到 7 月初这两周里，AI 头部厂商同时做出了三种截然不同的商业化选择：

厂商关键动作核心逻辑受众权衡
AnthropicFable 5 剥离订阅、usage credits 计费；Sonnet 5 全套餐可用、6 月 60% 降价用中低端模型补贴普及，用旗舰模型走按量付费牺牲重度 Agent 用户，留住企业 + Project Glasswing
OpenAI推理成本 ≥50% 下降；GPT-5.6 Terra 定价直接砍半；ChatGPT 匿名访客几百 GPU 兜住用软件层优化把订阅制承载能力翻倍保住 C 端订阅制的普及性；让企业客户享受 API 价格下降
DeepSeekV4 正式版 7 月中旬上线、首推峰谷定价（高峰 9-12 + 14-18 共 7 小时 ×2 倍）把边际算力成本按时段传导给用户激励用户错峰调用，提升整体资源利用率

数据来源：Anthropic 官方博客 / 智东西 7-2 / The Information 6-30 / OpenAI 定价页 / DeepSeek 官方公告 6-29 / 每日经济新闻 6-30。

三种答卷背后其实是同一个约束条件——推理算力供给的边际成本正在被三种方式对冲：Anthropic 通过分层剥离对冲、OpenAI 通过工程优化对冲、DeepSeek 通过时段套利对冲。哪一种能跑得最远，取决于三个变量：模型效果的边际收益、用户对复杂度的容忍度、监管对定价灵活度的许可。

短期看，OpenAI 的选择最讨好用户——订阅制不涨价、匿名访客都能用、Business API 降价。但代价是「工程 debt」——软件层优化到极限之后，下一次尖端模型来临时，OpenAI 必须走 Anthropic 的路。

Anthropic 的选择最诚实——承认订阅制承载不了尖端模型、承认要为最贵的算力单独计费。代价是短期社区口碑损失，以及给 OpenAI Codex 让出了一批高流失率用户。Sam Altman 在 5 月 14 日 Anthropic 首次公告 Agent SDK 剥离当天，就精准反手推出「新 OpenAI Business 用户两个月 Codex 免费」——这不是巧合，是竞对精准抢筹。

DeepSeek 的选择最中国化——不撕订阅制、不承认尖端模型问题、直接把「削峰填谷」的电网经济学搬到 AI 定价上。代价是高峰时段用户体验会明显下降、开发者要花额外精力做任务调度。但对 DeepSeek 而言，V4-Flash 已经连续 6 周霸占 OpenRouter 榜首、月调用量 4.66 万亿 Token（据每日经济新闻 6-29），峰谷定价的溢价空间足够撑起下一年的算力扩建。

六、监管与地缘：为什么 Fable 5 必须走「美国推理 1.1 倍」

剥离方案里有一条容易被忽视的细节：Fable 5 针对美国境内运行的工作负载，输入输出均按 1.1 倍价格计费，且推理过程仅在美国基础设施上运行。这条价格政策不是简单的地区加价，是 6 月 12 日美国出口管制指令的后续。

Fable 5 首发 3 天后就被美国出口管制全球下架，19 天后管制解除。管制期间发生了什么，Anthropic 与美国商务部没有公开细节，但从 6 月 30 日解除到 7 月 1 日回归中间只有一天间隔，说明双方在管制期间已经完成了「本地化推理 + 分级定价」的谈判。美国内计费加价 1.1 倍、只在美国基础设施推理——这份条款很可能是 Anthropic 为拿到解禁付出的对价。

这条政策对全球 AI 商业化的信号是：尖端模型不再是「云端全球可访问」，而是「按地区分级、按合规分层」。这个信号跟今天 AI News Daily 追踪的软银 SB Neo 10GW 俄亥俄州数据中心项目、Anthropic 探索三星 2 纳米自研芯片、韩国 AI 内存税收增量三件事拼在一起，说明「AI 的电力 + 算力 + 数据 + 合规」正在同时向本土化收拢——过去两年「一朵云跑全球」的假设，从 2026 下半年开始要重写。

Anthropic 内部对这份新格局其实早有准备。Claude Code 工程师 Thariq Shihipar 在 X 上回应「反蒸馏」争议时明确表示：「这是我们在 3 月推出的一项实验，目的是防止未经授权的转售商滥用账号，并防范模型蒸馏。」——反蒸馏机制、按地区推理、按模型剥离，三件事其实是同一份产业剧本的三个章节。

七、订阅制的临界点：SaaS 计费逻辑的第二次改写

把这条线拉到最长的时间维度，Anthropic 7 月 1 日的动作本质上是 SaaS 计费逻辑第二次被撕开。

第一次撕开是 2026-07-04 的微软 Cowork $30 + $0.01/credit（本站 7 月 4 日《微软的代理战争》已做深度分析）——微软把「席位」和「用量」拆成两条曲线，为 AI 代理场景建立独立计费。这一次是 Anthropic 更激进的一步——连「席位」都不给了，旗舰模型直接按 Token 计费。这两步加起来，AI 时代的 SaaS 计费终于有了完整的三层结构：

  • 普及层：席位订阅、基础模型免费或补贴（对应 Copilot Chat、Claude Sonnet 5、ChatGPT Free）——用来铺 baseline 现金流和用户习惯。
  • 专业层：席位订阅 + 用量计费（对应 M365 Copilot $30、Cowork $30 + $0.01/credit、Claude Max 20x + Agent SDK Credit）——覆盖企业代理的可预测负载。
  • 尖端层：纯按量计费、无席位补贴（对应 Fable 5 usage credits、GPT-5.6 Terra API、Anthropic Project Glasswing）——为最贵的算力和最独家的能力单独计费。

这套三层结构不是理论推演，是 2026 上半年 5 家头部厂商（Microsoft、OpenAI、Anthropic、Google、DeepSeek）实际定价页上的现实。每一层的现金流曲线都不一样：普及层是长期低毛利、专业层是中等毛利 + 高波动、尖端层是短期高毛利 + 高流失率。整个行业的收入结构会从过去的「订阅 90% + API 10%」，向未来的「订阅 40% + credit 30% + API 30%」平移。

结语：临界点不是终点，是选择开始的地方

7 月 1 日的 promotional access，看起来只是一份 7 天的促销条款；7 月 7 日之后，看起来只是一款模型退出订阅套餐。但如果把这份决定放到过去 18 个月推理成本曲线和过去 6 个月 Anthropic 商业模式变革的时间轴上，它其实是一个信号：大模型的订阅制补贴，走到了它的技术天花板。

OpenAI 用工程优化把这个天花板顶高了一次，Anthropic 用剥离旗舰承认这个天花板存在，DeepSeek 用峰谷定价把这个天花板的成本传导给用户。三种答卷都对——但它们说的不是同一个未来。

Fable 5 的 Token 消耗速度是 Opus 4.8 的 2 倍、单位任务账单是 4 倍，这份数字告诉每一个仍然依赖订阅制的开发者：接下来 12 个月，你为 AI 付出的每一美元，都会开始跟你「让 AI 干多少活」精确挂钩。这一步会痛，但也是 AI 从「体验品」走向「生产要素」的必经之路。

临界点不是终点。它是接下来 12 个月里，每一家 AI 厂商都会被迫做出选择的地方——你要做普及层？专业层？还是尖端层？Anthropic 已经给出了自己的答案。
